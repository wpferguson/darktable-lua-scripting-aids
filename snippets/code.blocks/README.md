# blocks of code to demonstrate how to do specific tasks

## collection_changed

Demonstrate how to use the `collection-changed` event

## communications_handler

Demonstrate how to use the `inter-script-communication` event

## grep

How to search for a string in a file

## install_module

How to install a library module in lighttable without crashing darktable.  This is necessary because starting darktable in darkroom mode and starting a lua script that installs a module will cause darktable to crash.
